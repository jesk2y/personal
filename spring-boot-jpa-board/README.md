## 프로젝트의 목적
> `spring-boot-jpa-board` 프로젝트의 기본 기능들을 JPA로 변환한다.
> 이를 통해 JPA의 기본적인 쓰임을 익힌다.
<br>

## 게시판의 기능
- 기본적인 CRUD
- 글 검색/페이징
- 댓글 페이징 기능
- 이미지 첨부 기능
